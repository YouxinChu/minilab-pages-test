Frequently asked questions

How is this different from CodeRabbit?

CodeRabbit reviews PRs without understanding your codebase. We index your repo first — architecture, patterns, past decisions — so every review is contextual. No generic "consider renaming this variable" noise.

Does it work with AI-generated code specifically?

Yes. We detect patterns unique to AI-generated code — confident-looking but misunderstood implementations, hallucinated design patterns, copy-paste logic that doesn't fit your architecture.

Will it slow down our CI pipeline?

Reviews run in parallel with your CI. Average review time: under 3 minutes for a typical PR.

What languages do you support?

JavaScript/TypeScript, Python, Java, Go, Rust at launch. More coming based on demand.

Can I customize what it flags?

Yes. Configure severity levels, ignore patterns, and teach it your team's conventions through a simple config file.
